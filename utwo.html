Featured Project

UTWO
Learning Resources, Free For All.

UTWO is a video-browsing platform built to help users discover learning resources through an immersive, modern interface. This locally ran project includes browsing, subscriptions, user profile functionality, search behavior, and so much more!

HTML CSS JavaScript UI Design Localized Project
Project: To create a polished learning-content platform built from the ground up.
Focus: Interface design, proper navigation flow, and a clean content experience.

Project Overview

UTWO was designed as a user-focused platform for discovering educational and technology-related video content through a sleek interface. The web application was originally created to satisfy as a project for my Human Computer Interaction course, and I completed the entire website in 3 months.

What I Built:

I developed the project structure, interface styling, browsing experience, and all aspects of the application. The goal was to create a platform that feels familiar, usable, and clean while presenting tech based educational content in a compelling way.

Current Presentation:

Since UTWO is currently run locally, this page serves as a polished case-study presentation using curated screenshots and a final walkthrough video to demonstrate the product’s design and functionality.

Interface Walkthrough

Below is a guided visual breakdown of UTWO’s main interface areas, showing how the platform supports browsing, search, subscriptions, profiles, and user guidance.

Homepage Experience

The homepage gives users a visually strong landing point, highlighting recommended educational content and giving them quick access to the core navigation areas.

Channel Page

The channel page organizes creator-specific content and presents the profile, banner, and featured videos in a layout that feels structured and easy to follow.

Profile Page

The profile page gives users a personalized view of their activity, saved content, and platform history while keeping the interface visually cohesive with the rest of the application.

Search Results

The search experience allows users to quickly locate content and visually scan results with a layout that prioritizes clarity, relevance, and readability.

Subscriptions View

This section helps users manage their subscribed channels through a card-based layout that keeps subscription information organized and easy to browse.

Help Center

The help page gives users guided visual instructions so they can understand platform functions quickly, reinforcing usability and accessibility within the experience.
